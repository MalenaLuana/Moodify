# 🎧 Moodify

**Moodify** es una aplicación de escritorio multiplataforma que detecta tu estado de ánimo y te recomienda música en base a tu energía emocional actual.

Desarrollado con:

- 🧠 **Node.js + TypeScript**
- 🖥️ **Electron**
- 🎵 **Spotify Web API**
